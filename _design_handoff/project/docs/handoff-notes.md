# NerdDen — заметки для реализации (handoff)

Накапливаю по ходу дизайна то, что важно учесть при переносе в код (SvelteKit + Tailwind v4 + PixiJS).

## Новый функционал, заложенный в дизайн

### Заметки / кандидаты (pencil marks) — НОВОЕ
- Кнопка **«Заметки»** переключает режим ввода.
- В режиме заметок цифра с нумпада добавляется маленькой пометкой в сетку 3×3 внутри клетки (а не как финальный ответ). Повторное нажатие той же цифры — убирает пометку.
- Вне режима заметок — цифра ставится как ответ (крупная, рукописная).
- Заметки рендерятся приглушённым серым (#8a8474, Hanken Grotesk), чтобы не спорить с ответами.
- В текущем коде проекта этого функционала нет — добавить:
  - в модель клетки поле `notes: Set<number>` / `boolean[9]`;
  - обработку ввода в режиме notes;
  - отрисовку мини-сетки кандидатов в `SudokuBoard` (PixiJS — мелкий текст 3×3);
  - кнопку-тоггл «Заметки» с активным состоянием.
